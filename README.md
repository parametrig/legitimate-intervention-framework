# Legitimate Intervention Framework (LIF)

An open-source research framework for onchain governance and protocol safety.

## Overview
This repository contains the data pipeline, research methodology, and manuscript for the Legitimate Intervention Framework (LIF). The project quantifies and analyzes systemic risk in decentralized finance, proposing automated "Ex Ante" (Circuit Breaker) and "Ex Post" (Governance Freeze) mechanisms.

**Key Stats (Jan 2026):**
- **718 exploits** analyzed (2014-2026)
- **$80.48 billion** in total losses
- **424 LIF-addressable** incidents ($10.21B)
- **120 unique intervention cases** (combined datasets, deduplicated)
- **$2.77B** in losses prevented (27.2% capture rate)

**Market Analysis:**
- **Coverage:** 80.6% of at-risk capital saw intervention attempts
- **Effectiveness:** 26.5% of covered risk successfully saved
- **Gap:** $7.70B incurred despite intervention — the ROI opportunity

**Intervention Datasets:**
- `lif_all_interventions.csv` - 114 standardized DeFi intervention cases
- `lif_intervention_metrics.csv` - 30 highly curated cases with detailed timing/response analysis
- Combined unique interventions: **120 cases** (after deduplication)

**Recent Additions (Jan 2026):**
- Complete data refresh with corrected `loss_prevented_usd` values
- Added 13 cases with updated metrics (BNB, Ronin, Cetus, DAO Fork, etc.)
- Dataset now spans 2014-01-01 to 2026-01-21

## Quick Start

### Prerequisites
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### Update Statistics
```bash
python3 scripts/core/update_stats.py
```

### Visualizations
Run the v1 notebook to generate the curated visual set and overwrite stats:
```bash
jupyter nbconvert --execute --to notebook scripts/analysis/lif_charts_v1.ipynb
```

Legacy figures are archived under `visualizations/archived/v0_legacy/` and generated by the archived notebook.

**Outputs:**
- `visualizations/v1/*.png`
- `visualizations/archived/v0_legacy/*.png`
- `data/refined/lif_stats.json` (rich statistics with sources)
- `data/refined/lif_intervention_metrics.csv` (intervention effectiveness subset)
